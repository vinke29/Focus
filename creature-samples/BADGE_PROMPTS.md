# Kokoon Badge Art Prompts

Use these prompts in ChatGPT image generation. Save each PNG as `badge_{id}.png` (e.g., `badge_first_session.png`, `badge_streak_3.png`).

**After generating:** give all the PNGs to Claude to process (remove background, resize, swap into the app).
