# Kokoon Evolved Character Art Prompts

Use these prompts in ChatGPT image generation. Save each PNG as `{id}_evolved.png` (e.g., `karasu_evolved.png`) into a folder.

**After generating:** give all the PNGs to Claude to process (remove background, resize to 320×380, swap into the app).

**Key differences from base prompts:**
- Same character, grown up: taller proportions, no longer chibi
- More majestic, powerful, detailed features
- Stronger magical aura, flowing elements, enhanced signature traits
- Same accent color, same art style, same background
