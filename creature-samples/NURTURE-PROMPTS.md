# Kokoon Nurture Art Prompts

Use these prompts in ChatGPT image generation. Save each PNG as `{id}_nurture.png` (e.g., `shiro_nurture.png`) into a folder.

**After generating:** give all the PNGs to Claude to process (remove background, resize to 280×320, swap into the app).

**Key differences from base prompts:**
- Character is holding / cradling / sheltering a large luminous egg
- Egg is cream-white, slightly larger than the character's head, glowing warmly from within in the character's accent color
- Expression is tender, protective, focused — they are guarding something precious
- Same chibi proportions, same art style, same background as base characters
